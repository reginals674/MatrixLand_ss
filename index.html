Fuente de Energía

Tutoriales de Operaciones

Aprende paso a paso cómo se resuelve cada operación con matrices

Para sumar matrices se suman los elementos que están en la misma posición (misma fila y columna). Ambas matrices deben tener el mismo tamaño.

La resta de matrices funciona igual que la suma, pero restando el elemento de B al elemento de A en la misma posición.

La multiplicación por escalar significa multiplicar cada elemento de la matriz por el mismo número k.

Para multiplicar matrices, cada elemento del resultado se obtiene multiplicando una fila completa de A por una columna completa de B, elemento a elemento, y sumando esos productos. El número de columnas de A debe ser igual al número de filas de B.

La transpuesta convierte las filas en columnas (y las columnas en filas). El elemento en la posición (i,j) pasa a la posición (j,i).

El determinante es un número asociado a una matriz cuadrada. A diferencia de una matriz normal, no se escribe entre corchetes, sino entre barras verticales: |A|.

¿Cómo Jugar?

Vidas

Empiezas con 3 vidas. Cada respuesta incorrecta te quita una. Si llegas a 0, la partida termina.

Consigue una racha de 5 aciertos seguidos y recuperas 1 vida (hasta el máximo de 3).

Racha

Cada acierto consecutivo aumenta tu racha. Una respuesta incorrecta la reinicia a 0.

Cada punto de racha añade +10% de puntos extra al siguiente acierto.

Puntos

Cada nivel otorga puntos base al resolver bien un desafío:

  • Básico10 pts
  • Intermedio20 pts
  • Avanzado30 pts
  • Retos Extra40 pts

Pasos para jugar

  1. Elige un nivel según tu experiencia.
  2. Observa la operación planteada entre las matrices (o el escalar).
  3. Arrastra los números correctos desde la "Fuente de Energía" hacia la matriz de respuesta.
  4. Pulsa "Verificar Secuencia" para comprobar tu resultado.
  5. Usa el botón de Tutoriales si necesitas repasar cómo se resuelve cada operación.
